# Game Update Manager

**Веб-интерфейс для отслеживания и управления обновлениями компьютерных игр в локальной сети компьютерного клуба**

## Описание проекта

Система предназначена для автоматизации процесса обновления игровых клиентов и программного обеспечения в компьютерных клубах. Работает в изолированной локальной сети (LAN), не требует постоянного доступа к интернету.

### Решаемые проблемы

- **Отсутствие единой точки контроля** — быстрый обзор состояния всех ПК клуба
- **Ручное обновление на каждой машине** — централизованное управление патчами
- **Невозможность быстрого отката** — функция rollback к стабильной версии
- **Отсутствие гибкого управления** — выбор ПК, версий и времени обновления

### Основные возможности

1. Сканирование установленных игр и сравнение версий с эталонными данными
2. Формирование отчётов о необходимости обновлений
3. Централизованное управление задачами обновления
4. Откат к предыдущим версиям игр (rollback)
5. Адаптивный веб-интерфейс для доступа с любого устройства в сети

## Стек технологий

### Backend
- **PHP 8.x** — серверная логика и API
- **Apache** — веб-сервер (в составе XAMPP)
- **MySQL/MariaDB** — база данных для хранения эталонных версий, логов и конфигураций

### Frontend
- **React 18+** — компонентный UI
- **JavaScript (ES6+)** — язык программирования
- **Vite** — сборка и dev-сервер

### Инструменты
- **Git** — система контроля версий
- **Postman** — тестирование API
- **GitHub/GitLab** — хостинг репозитория

### Основные модули

**Frontend (React):**
- Отображение списка ПК и статусов обновлений
- Формы управления задачами
- Визуализация отчётов

**Backend (PHP):**
- `ScannerService` — опрос клиентских машин
- `VersionManager` — сравнение версий, определение необходимости обновления
- `TaskScheduler` — планирование и очередь задач
- `RollbackHandler` — восстановление предыдущих версий
